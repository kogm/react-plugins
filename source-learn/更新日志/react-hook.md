## 动机

```
Hook 解决了我们五年来编写和维护成千上万的组件时遇到的各种各样看起来不相关的问题。
无论你正在学习 React，或每天使用，或者更愿尝试另一个和 React 有相似组件模型的框架，你都可能对这些问题似曾相识。

在 React DevTools 中观察过 React 应用，你会发现由 providers，consumers，
高阶组件，render props 等其他抽象层组成的组件会形成“嵌套地狱”。
尽管我们可以在 DevTools 过滤掉它们，但这说明了一个更深层次的问题：React 需要为共享状态逻辑提供更好的原生途径。
```
